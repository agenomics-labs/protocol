# ADR-099 — @agenomics/idl SDK Package

## Status

Superseded by ADR-099-idl-package

## Date

2026-04-23

## Context
Program IDs are duplicated across mcp-server, indexer, x402-relay, and tests.
A single cluster-keyed program ID manifest eliminates drift.

## Decision
Create `sdk/idl/` publishing as `@agenomics/idl`. Provides:
- Cluster-keyed program ID manifest: `{ devnet: {...}, mainnet: {...} }`
- TypeScript types re-exported from `target/types/` (post `anchor build`)
- `getProgramIds(cluster: "devnet"|"mainnet")` helper

## Consequences
- All services import program IDs from one place.
- `target/types/` must be present (generated by `anchor build`).
