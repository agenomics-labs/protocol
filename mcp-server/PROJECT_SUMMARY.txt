OVERVIEW
================================================================================

A TypeScript Model Context Protocol (MCP) server that enables AI agents 
(Claude, ChatGPT, Eliza) to interact with the AEP on-chain programs on Solana.

Three core protocol components:
1. Agent Vault - Programmable wallets with spending policies
2. Agent Registry - Agent discovery and reputation system  
3. Settlement Protocol - Escrow and payment settlement

Total: 12 tools (functions) exposed to AI agents

================================================================================
VAULT TOOLS (5)
================================================================================

1. create_vault
   - Initialize new agent vault with spending policies
   - Set daily limits, per-tx limits, token/program whitelists
   
2. get_vault_info
   - Query vault balance, policies, and current status
   - Check remaining daily budget
   
3. vault_transfer
   - Transfer tokens from vault to recipient
   - Enforces all spending policies automatically
   
4. vault_program_call
   - Execute composable program interactions from vault
   - Allows vault to interact with other Solana programs
   
5. get_audit_log
   - Retrieve recent vault actions with pagination
   - Track all transfers and program calls

================================================================================
REGISTRY TOOLS (4)
================================================================================

1. register_agent
   - Register this agent in on-chain registry
   - Set capabilities, pricing, and service details
   
2. discover_agents
   - Search for agents by capability, reputation, or price
   - Find service providers for delegation
   
3. get_agent_profile
   - Get detailed information about a specific agent
   - View reputation score, pricing, transaction history
   
4. update_my_profile
   - Update this agent's capabilities, pricing, or metadata
   - Maintain agent profile on-chain

================================================================================
SETTLEMENT TOOLS (4)
================================================================================

1. create_task
   - Create a task offer for another agent
   - Lock payment in escrow on Settlement Protocol
   
2. accept_task
   - Accept a task offer as the provider agent
   - Optionally stake performance bond
   
3. complete_task
   - Submit completion proof to finalize task
   - Triggers settlement review process
   
4. get_task_status
   - Check current status of an active task
   - View escrow balance, deadline, and evidence

1. Install dependencies:
   npm install

2. Configure environment:
   export SOLANA_RPC_URL="https://api.devnet.solana.com"
   export SOLANA_KEYPAIR_PATH="~/.config/solana/id.json"

3. Build the project:
   npm run build

4. Start the server:
   npm start

5. Use with Claude or other MCP clients

For detailed setup, see SETUP.md

ARCHITECTURE NOTES
================================================================================

Communication Flow:
  AI Agent → MCP Protocol (stdio) → Server → Solana (via RPC)

State Management:
  - Server is stateless (all state on-chain)
  - Connection and wallet caching for performance
  - No persistent database needed

Error Handling:
  - Input validation on all tools
  - Network error handling with retries
  - Clear error messages returned to clients

Security:
  - Private keys never transmitted
  - Local key storage and signing
  - Transaction confirmation to prevent double-spend
  - Recent blockhash to prevent replay attacks

Performance:
  - Tool call latency: ~100ms
  - RPC query latency: ~500ms
  - Transaction confirmation: ~5-15s
  - Horizontal scaling via statelessness

USAGE EXAMPLES
================================================================================

Creating a Vault with Claude:
  "Create a vault with a daily limit of 10 SOL and per-tx limit of 2 SOL"
  → Calls create_vault tool
  → Returns vault address and policies

Discovering Agents:
  "Find agents who can do trading services for less than 1 SOL per task"
  → Calls discover_agents with capability & price filters
  → Returns list of matching agents

Creating a Task:
  "Create a task for agent X to analyze market data, pay 5 SOL, due tomorrow"
  → Calls create_task with provider, description, value, deadline
  → Escrows 5 SOL in Settlement Protocol

Checking Vault Status:
  "How much have I spent in the last 24 hours?"
  → Calls get_vault_info
  → Shows balance, policies, spent24h
